# caseware


To run this Angular project on your local machine, follow these steps:

Clone the Repository
Open your terminal and clone the repository:

bash
git clone https://github.com/yourusername/product-app.git
Replace the URL with the correct repository URL.

Navigate to the Project Directory
Change into the project folder:

bash
cd product-app
Install Dependencies
Use npm to install all required dependencies:

bash
npm install
Run the Development Server
Start the Angular development server with:

bash
ng serve
By default, the application will be available at http://localhost:4200.
